Download et Installation des polices Gentiumalt

Pour Windows, Macintosh, Linux et Openoffice




Installation des polices Gentiumalt sous Windows :

Pour Window, vous pouvez les downloader ici. ( Gentium Release 1.01 (Windows and other) ZIP archive, 600 Kb)

- Vous les décompressez et vous les copiez/collez dans le dossier : C:\WINDOWS\Fonts.
- Vous relancez votre ordinateur et vous les voyez dans votre liste de polices.


Installation des polices Gentiumalt sous Mac :

Pour Mac, vous pouvez les downloader ici. (Release 1.01 (Macintosh) Zip archive, 543 Kb). Vous les décompressez, puis :

- Sous Mac OS X, vous copiez/collez les fichiers des polices (Gentium, GentiumAlt) dans le dosssier ~/Bibliothèque/Fonts
A partir de Os X.3 ou supérieur, dans le dossier Applications se trouve le logiciel Fontbook (livre des polices en français) qui permet de gérer les typos à installer. Plus de renseignements ici.

- Sous Mac OS 9, vous placez les polices sur le dossier sustème fermé, un message demande si voulez les installer (remerciements au forum Macbidouille).


Installation des polices Gentiumalt sous Linux :

Pour Linux, vous pouvez les downloader ici.

- Ouvrir l'installateur de polices, passer en super-utilisateur, l'encadré rouge autour de l'installateur de polices indique qu'on est en mode super-utilisateur.
- Dans la barre des menus, créer un nouveau dossier appelé polices_gentium par exemple, ouvrir ce nouveau dossier qui pour le moment est vide, puis cliquer juste à côté sur l'outil "Ajouter des polices", choisir l'emplacement où a été déposée la police, la sélectionner, valider ( remerciements à Magnificus pour le topo).

- Si cette méthode ne convient pas à la distribution Linux que vous possédez, vous trouverez plus de détails en Anglais, ici.


Installation des polices Gentiumalt sous Openoffice :

Pour Openoffice, vous pouvez les downloader ici.

- Taper la commande : /usr/lib/openoffice/spadmin
- Cliquer sur le bouton Fonts
- Sélectionnez le dossier dans lequel vous avez placé les fichiers downloadés et décompressés
- Les polices seront copiées dans le dossier ~/openoffice/user/fonts, dans le répertoire fonts.dir créé automatiquement.
